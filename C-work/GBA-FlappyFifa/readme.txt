welcome to FIFA GBA

Story:
this is a FIFA/pong/flappy bird cross over where you are playing against flappy and the 4 corners. Flappy has taken your cat as hostage and the only
way to rescure your cat is to win the World Cup championship against flappy (aka Pong Cup). If you lose, flappy gets to send your cat to the
shadow realm.


HOW TO PLAY:
ENTER to START GAME
BACKSPACE to go back to the BEGINNING of the start game in any state
How to move your red pong foot: LEFT/RIGHT KEYS 

START GAME DEFAULT -> (PRESS ENTER) -> PLAYING INITIALIZED -> (press L/R to collide ball with foot) -> get score up to 6 to win, score 0 to lose.
