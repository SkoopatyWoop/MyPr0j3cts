Racknerd Review: is Racknerd VPS worth the buy?

  • January 1, 2023
  • ReviewHosting

Overview

I've been using Racknerd's services for over a month now and I wanted to write a review discussing a few topics: My experience with the Racknerd vps including a performance test for my usecase, its general uses & a breakdown of wether or not Racknerd is right for you.

My Experience

I came across Racknerd when I was looking for a place to host my website. I was thinking of using Hostinger's shared hosting until I came across some Reddit posts bashing how awful Hostinger's service is (link). I will be discussing why I chose Racknerd over Hostinger later on.

I'd like to state my general experience first. To simply put it, Racknerd gets the job done. I haven't had anny issues with the vps shutting down on me or having any throttling concerns. My website seems to be up 24/7 and that's all I could ask for. A major plus was that I bought Racknerd vps for a great deal and I am paying a very fair price for the service. If you read this article and would like to check it out, I have provided a a link below for the deal I signed up for. Now let's talk about the setup process.

Setup Process

When I first bought the vps, I immediately got my login details for root access to the vps from Racknerd. I am not a vps connosieur so I looked up a quick tutorial on how to "set up racknerd vps", and it was basically smooth sailing from there. The setup process included logging in to racknerd with my account credentials, picking which Linux server I wanted to use, then ssh into my vps to set up cloud panel. From there, I had the Cloud Panel interface available on my browser, allowing me to host Wordpress, Static, PHP, Node.js, & Python sites.

Now, I will finish with this: the setup process isn't the easiest on the market, but it's easy to find tutorials.

Performance

After testing my website, here are the Performance benchmarks from PageSpeed insights

  • Website Performance
    • Mobile Score: 91/100
    • Desktop Score: 91/100

I want to share that this website is not optimized for performance, and I am still working on it. So the great performance score gives me confidence in Racknerd's product.

Usecases for Racknerd vps

As mentioned, I am using racknerd to host my website called Axelorate. I am making changes continually and I find the Cloud Panel GUI to be pretty useful when making updates. My only complaint is that I haven't found a tutorial to link my Github repository to Cloud Panel so I wouldn't have to drag/drop my files each time I decide to publish content.

However, hosting a website is only one usecase for having a VPS. You could host your own VPN or create a gaming server if you'd like (more info)! This leads me to a short summarization of the difference between shared hosting Vs VPS and why I chose Racknerd.

Shared Hosting vs VPS hosting: Is Racknerd VPS right for you?

Shared Hosting is like a prebaked pizza that is made for the general population. Most people like a nice Parmasean Pizza so let's use that as the example. Shared Hosting is preconfigured for you and all you need to do is provide your website files and you're ready to go! However, you have to share your delicious server with other website owners, and on top of that, you don't have root access to the server. This is where VPS comes in: you have root access, your own dedicated resources, and access to the back end. Secondly, if you have the expertise, you can make whatever pizza you'd like! (I'm a barbeque fan).

To summarize, if you're a beginner and just want to put your website up with no extra usecases and don't mind sharing a server, use Shared Hosting. Otherwise, Racknerd could be right for you :). I have a link for a sweet deal on a Racknerd VPS here.
